# Do Not Do

- Non cambiare stack tecnologico.
- Non introdurre backend o database senza richiesta.
- Non aggiungere librerie UI pesanti.
- Non spezzare parole in board layout salvo requisito esplicito contrario.
- Non modificare formattazione in aree non correlate.
- Non eliminare file o cartelle senza conferma.
- Non deployare, pubblicare o pushare senza autorizzazione finale.
