# UI & Motion Rules

- Usa sfondo globale blu notte `#020b2d` o equivalente vicino.
- Usa bordi/accenti neon ciano e viola.
- Mantieni contrasto alto e leggibilità TV-first.
- Le tessere devono avere un flip 3D sull'asse Y pulito e fluido.
- Le animazioni non devono rompere il layout né causare flicker visibile.
- Per mobile e desktop, preserva le proporzioni visive principali del board.
- Evita effetti gratuiti che riducano chiarezza o performance.
