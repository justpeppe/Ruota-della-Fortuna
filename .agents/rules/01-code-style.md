# Code Style

- Preferisci funzioni pure in `src/utils/`.
- Tieni i componenti visuali focalizzati sul rendering.
- Usa props esplicite; evita oggetti anonimi profondi se non servono.
- Commenta il perché, non il cosa.
- Mantieni i file sotto ~300 righe quando possibile.
- Non introdurre astrazioni premature.
- Evita refactor strutturali non richiesti dal task.
