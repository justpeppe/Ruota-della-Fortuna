# UI Polish & Verify

## Goal
Rendere il layout coerente con l'estetica game show e verificabile visivamente.

## Procedura

1. Identifica gli elementi ad alto impatto visivo:
   - sfondo
   - cornici neon
   - tabellone
   - hint box
   - player cam badge
   - flip tiles

2. Applica modifiche piccole e mirate.

3. Controlla leggibilità:
   - contrasto testo/sfondo
   - spessori bordi
   - dimensioni font
   - spaziature

4. Verifica comportamento responsive:
   - desktop wide
   - laptop
   - mobile landscape
   - mobile portrait, se supportato

5. Produci checklist finale:
   - layout non collassa
   - board centrata
   - video 9:16 rispettato
   - animazione flip fluida
   - hint leggibile
