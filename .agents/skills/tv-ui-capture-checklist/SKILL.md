# TV UI Capture Checklist

## Goal
Fornire una review visuale ripetibile e orientata alla demo.

## Instructions

1. Controlla gerarchia visiva.
2. Controlla allineamenti e proporzioni.
3. Controlla leggibilità testo e badge.
4. Controlla coerenza colori e glow.
5. Controlla transizioni e motion.
6. Restituisci una checklist manuale semplice da seguire.

## Output format
- criticità bloccanti
- criticità minori
- checklist di verifica
- suggerimenti di polish
